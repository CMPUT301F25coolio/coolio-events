# Object-Oriented Analysis: CRC Cards

## Entrant
### Responsibilities:
- Has profile
- Maintains list of their events
- Search for events
- View event details
- Entering the wait list for an event
- Register or decline an event, or cancel registration
- Receives notifications 
- Scan QR code
### Collaborators:
- Profile
- Search
- Waitlist
- Event
- EventList
- Notification
- Camera 

## Profile 
### Responsibilities:
- Stores Name
- Stores Email
- Stores password
- Stores phone number (optional)
### Collaborators:
- Entrant
- Organizer
- Administrator
